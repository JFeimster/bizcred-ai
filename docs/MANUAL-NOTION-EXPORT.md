# Manual Notion Export Guide

BizCredit OS can prepare local files for manual import into a Notion workspace. This flow does not require a backend.

## Package Workflow

1. Open the React app.
2. Go to Exports.
3. Select Generate Notion Package.
4. Save the downloaded files.

The package may include:

- `bizcredit-passport.md`
- `tradeline-planner.csv`
- `vendor-directory.csv`
- `roadmap.md`
- `import-instructions.md`

## Database Imports

For CSV files, open the target Notion database and use its CSV import or merge option.

## Page Imports

For Markdown files, create or open a page and use Notion Markdown import.

## Privacy Note

All generation and export happens locally in the browser.
