# Privacy and Safety Rules

## Privacy

The React app scaffold stores profile, planner, and roadmap state in browser localStorage only. There is no auth layer, no backend storage, and no user tracking in this phase.

## Safety language

Use these frames:

- educational planning
- readiness
- setup signals
- common gaps
- next steps
- verify directly

Avoid these claims:

- approval guarantees
- funding guarantees
- bureau reporting guarantees
- tradeline guarantees
- score increase guarantees
- credit repair framing
- lender-approved or pre-approved claims

## Vendor data

Vendor examples and saved planner items are research aids. Users should verify vendor requirements, terms, and reporting directly before relying on them.
